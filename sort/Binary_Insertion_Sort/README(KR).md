### 이진 삽입 정렬
--------
우선 삽입 정렬은, 한 번에 한 요소의 위치를 최종적으로 고정시키는 간단한 정렬 알고리즘이다. 

한번 반복할 때마다 입력 목록에서 한 요소를 사용하고, 정렬된 목록 내에서 삽입될 위치를 찾아 해당 요소를 추가한다.
추가된 요소는 입력 목록에서 삭제되며, 입력 요소가 남아있지 않을 때 까지 반복한다.

Quicksort, merge sort와 같은 고급 알고리즘과 비교했을 때 크기가 큰 목록에서는 비효율적이지만, 간단하게 구현이 가능하고, 크기가 작은 목록에서 효율적이고, 다른 단순한 알고리즘보다 더 효율적이라는 장점이 있다.

이진 삽입 정렬은 삽입정렬을 개선한 알고리즘으로, 이진 검색을 이용하여 새 요소를 삽입할 위치를 결정한다.

이때 요소를 삽입하기 위해 교환이 일어나기 때문에 이진 삽입 알고리즘은 평균적으로 O(n^2)의 실행 시간을 가진다.

하지만 여러 요소의 위치를 계산한 뒤에 이동을 하면 교환의 횟수를 줄일 수 있다.

두 요소의 목표 위치가 올바른 위치로 이동하기 전에 계산된 경우,
무작위 데이터의 경우 스왑 수를 약 25% 줄일 수 있다. 극단적인 경우, 이 변형은 병합 분류와 유사하게 작동한다.
