# 2D Matrix transpose
  transposes 2d matrices
```js
let matrix = [
  [1, 1, 1],
  [2, 2, 2],
  [3, 3, 3]
 ]
```
this is transposed to
```js
let transposed = [
  [1, 2, 3],
  [1, 2, 3],
  [1, 2, 3]
]
```
## Running
```bash
node matrix_2d_transpose.js
```
## Tests implemented with [jest](https://jestjs.io/)
